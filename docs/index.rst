Welcome to the Forensics Artifacts Knowledge Base
=================================================

The Digital Forensics Artifacts Repository, is a free, community-sourced,
machine-readable knowledge base of digital forensic artifacts that the world
can use both as an information source and within other tools.

This knowledge base provides documentation accompanying the machine-readable
definitions in the repository.

The source code is available from the `project page <https://github.com/ForensicArtifacts/artifacts-kb>`__.
